# Capstone Project
This repository contains the frontend and the backend code for the Capstone Project.
# How to start working?
1.Clone Repository Using :
```bash
https://github.com/DaivPP/Capstone-Project.git
```
2. For Frontend -

```bash
cd client
```
3. For Backend -

```bash
cd server
```
4. Install the dependencies

```bash
npm i
```

5. Run the code

```bash
npm run dev
```
